## 流程图预览

```mermaid
graph TD
    subgraph "阶段一: 任务接收与分派"
        A1[上级部门通过微信/QQ下发任务] --> A2[村/社区两委接收]
        A2 --> A3{是否需要实地采集?}
    end
    subgraph "阶段二: 数据采集"
        A3 -- 是 --> B1[分派给网格员/小组长]
        B1 --> B2{采集方式?}
        B2 -- 主流方式 (纸质) --> B3[打印空表后, 入户/电话手写记录]
        B2 -- 高级用户 (在线) --> B4[使用腾讯文档在线协作填写]
        B2 -- 简单信息 (微信) --> B5[微信群接龙/文字上报]
    end
    subgraph "阶段三: 数据整合与处理 (核心瓶颈)"
        A3 -- 否 --> C0[村委干部直接利用现有数据]
        B3 --> C1[手动录入: 将手写纸质信息敲入电脑]
        B4 --> C2[数据导出: 从在线文档下载Excel]
        B5 --> C3[手动整理: 复制粘贴微信群信息]
        C0 --> C4[数据整合]
        C1 --> C4
        C2 --> C4
        C3 --> C4
        C4 --> C5[数据匹配: 从户籍总表/其他母表查找并补充基础信息]
        C5 --> C6{操作者Excel技能水平?}
        C6 -- 高级 (少数) --> C7[使用VLOOKUP等函数自动匹配]
        C6 -- 普通 (多数) --> C8[手动逐条复制粘贴]
        C7 --> C9[完成数据整理]
        C8 --> C9
    end
    subgraph "阶段四: 审核与上报"
        C9 --> D1[内部审核]
        D1 --> D2{是否需要纸质盖章?}
        D2 -- 否 (主流) --> D3[通过微信/QQ提交电子版]
        D2 -- 是 (少数) --> D4[打印/盖章/扫描后提交]
        D3 --> D5[上报完成]
        D4 --> D5
    end
    subgraph "阶段五: 归档与复用"
        D5 --> E1[数据存档]
        E1 --> E2[保存在分管干部个人电脑]
        E2 --> E3[形成数据孤岛, 复用困难]
    end
